# 🏥 Vaidya - Telemedicine Healthcare Platform

A comprehensive telemedicine platform built with Python (Flask) for the backend and vanilla JavaScript (HTML/CSS/JS) for the frontend.

## ✨ Features

- 🔐 User authentication (Patients & Doctors)
- 👤 Patient dashboard with health records, vitals, and appointments
- 👨‍⚕️ Doctor dashboard with patient management and consultations
- 💊 Prescription management system
- 📅 Real-time consultation scheduling
- 📊 Health analytics and vitals tracking
- 🔒 Secure JWT-based authentication

## 🛠️ Tech Stack

- **Backend:** Python, Flask, Flask-CORS
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla), Tailwind CSS
- **Authentication:** JWT (JSON Web Tokens) via PyJWT
- **Persistence:** In-memory storage (demo). Can be upgraded to MongoDB/Postgres.
- **Deployment:** Render

## 🚀 Getting Started

### Prerequisites
- Python 3.10+

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/srikiran1905s/FEDF.git
   cd FEDF
   ```

2. **Install backend dependencies**
   ```powershell
   py -3 -m venv .venv
   .\.venv\Scripts\Activate.ps1
   pip install -r backend_flask/requirements.txt
   ```

3. **Set up environment variables**
   Create `.env` file in backend directory:
   ```env
   MONGODB_URI=your_mongodb_connection_string
   JWT_SECRET=your_secret_key
   PORT=5000
   NODE_ENV=development
   ```

4. **Seed the database**
   ```bash
   npm run seed
   ```

5. **Start the backend server**
   ```powershell
   python backend_flask/app.py
   ```

6. **Start the frontend server** (in a new terminal)
   ```bash
   cd ../html-frontend
   python -m http.server 8080
   ```

7. **Access the application**
   - Frontend: open `html-frontend/index.html` (or `python -m http.server 8080`)
   - Backend API: http://localhost:5000

## 📦 Deployment to Render

For detailed deployment instructions, see [DEPLOYMENT.md](./DEPLOYMENT.md)

### Quick Deploy Steps

1. **Backend (Flask Web Service)**
   - Build Command: `pip install -r backend_flask/requirements.txt`
   - Start Command: `python backend_flask/app.py`
   - Health Check: `/api/health`

2. **Environment Variables**
   - `PORT=5000`
   - `FRONTEND_URL=https://<your-frontend>.onrender.com`
   - `JWT_SECRET` (autogenerated or set manually)
   - `MONGODB_URI` (optional; if set, enables persistence)

3. **Frontend** as Static Site
   - Root Directory: `html-frontend`
